How to add public music tracks:

1. Put your audio files in this folder.
   Example: duality.mp3

2. Open tracks.js and add one item per track:
   {
     title: "Duality",
     artist: "CATE",
     duration: "04:20",
     year: "2026",
     type: "Original",
     src: "assets/music/duality.mp3",
   },

3. Open index.html. Visitors will click a track in the playlist to listen.
